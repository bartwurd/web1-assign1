Brennan's Favourite Tags

Anchor Tag
The anchor tag <a></a> allows you to create links to things, be it on your site, on another site or even on the same page. I like that you can add images inside of anchor tags to create image links.
Image Tag
The image tag <img></img> allows you to add images to your page. they use two attributes, src and alt. src denotes where the image is located and alt is for writing a description of the image
List Tag
There are three different list tags, unordered list <ul></ul>, ordered list <ol></ol> and descriptive list <dl></dl>. Unordered lists are lists where the order of the items doesn't matter. Ordered lists are lists where the order of the items does matter. Descriptive lists are lists where you can write a list item and then a description of it, like I'm doing with this list of tags.
Heading Tags
Heading tags differentiate headings from the rest of the text. Heading tags have a hierarchy of 1-6 with 1 being the biggest and most important and 6 being the smallest and least important. Heading tags are formatted as <h1></h1> but the number (in this case 1) can be any number from 1-6 depending on desired size.
Important Text Tag
The important text tag <strong><strong> is for emphasizing text by making bold
